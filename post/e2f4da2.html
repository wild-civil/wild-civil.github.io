еӯҰзү©иҒ”зҪ‘д№ӢESP8266NodeMCUпјҲдә”пјү

дёҖгҖҒESP8266й—Әеӯҳж–Үд»¶зі»з»ҹеҹәжң¬ж“ҚдҪң

еҪ“жҲ‘д»¬дёҠдј зЁӢеәҸз»ҷESP8266ж—¶пјҢжҲ‘д»¬зҡ„зЁӢеәҸе…·дҪ“еӯҳж”ҫеңЁд»Җд№Ҳең°ж–№е‘ўпјҹ

жҜҸдёҖдёӘESP8266йғҪй…ҚжңүдёҖдёӘй—ӘеӯҳпјҢиҝҷдёӘй—ӘеӯҳеҫҲеғҸжҳҜдёҖдёӘе°ҸзЎ¬зӣҳпјҢжҲ‘д»¬дёҠдј зҡ„ж–Үд»¶е°ұиў«еӯҳж”ҫеңЁиҝҷдёӘй—ӘеӯҳйҮҢгҖӮиҝҷдёӘй—Әеӯҳзҡ„е…Ёз§°жҳҜSerial Peripheral Interface Flash File SystemпјҲSPIFFSпјүгҖӮ

йҷӨдәҶеҸҜд»Ҙеӯҳж”ҫдёҠдј зҡ„зЁӢеәҸд»ҘеӨ–пјҢжҲ‘д»¬иҝҳеҸҜд»Ҙе°ҶзҪ‘йЎөж–Үд»¶жҲ–иҖ…зі»з»ҹй…ҚзҪ®ж–Үд»¶еӯҳж”ҫеңЁESP8266зҡ„й—ӘеӯҳдёӯгҖӮеңЁиҝҷиҠӮиҜҫйҮҢпјҢжҲ‘д»¬е°ҶеӯҰд№ еҰӮдҪ•еҲ©з”ЁзЁӢеәҸеҜ№й—Әеӯҳж–Үд»¶зі»з»ҹпјҲSPIFFSпјүиҝӣиЎҢж–Үд»¶иҜ»еҸ–е’Ңдҝ®ж”№гҖӮ

1гҖҒйҖҡиҝҮзЁӢеәҸеҗ‘й—Әеӯҳж–Үд»¶зі»з»ҹеҶҷе…ҘдҝЎжҒҜ

еңЁдҪҝз”ЁSPIFFSеӯҳеӮЁж–Үд»¶д»ҘеүҚпјҢжҲ‘д»¬еҝ…йЎ»дҪҝз”Ё#include "FS.h",еҰӮд»ҘдёҠзЁӢеәҸз¬¬18иЎҢжүҖзӨәгҖӮ

зЁӢеәҸз¬¬20иЎҢпјҢжҲ‘д»¬е»әз«ӢдәҶдёҖдёӘеӯ—з¬ҰдёІеҸҳйҮҸгҖӮиҜҘеҸҳйҮҸз”ЁдәҺеӯҳеӮЁж–Үд»¶дҪҚзҪ®е’Ңж–Үд»¶еҗҚгҖӮе…¶дёӯ/д»ЈиЎЁж №зӣ®еҪ•гҖӮ/taichi-maker/д»ЈиЎЁж №зӣ®еҪ•дёӢзҡ„taichi-makerзӣ®еҪ•гҖӮnotes.txtд»ЈиЎЁзқҖж–Үд»¶еҗҚз§°гҖӮ

зЁӢеәҸз¬¬27иЎҢSPIFFS.format();жҳҜеҜ№й—Әеӯҳж–Үд»¶зі»з»ҹиҝӣиЎҢж јејҸеҢ–гҖӮиҝҷеҫҲжғіжҳҜжҲ‘д»¬еҜ№uзӣҳиҝӣиЎҢж јејҸеҢ–зҡ„ж“ҚдҪңгҖӮжӮЁж— йңҖжҜҸж¬ЎдҪҝз”Ёй—Әеӯҳж–Үд»¶зі»з»ҹйғҪеҜ№е®ғиҝӣиЎҢж јејҸеҢ–ж“ҚдҪңгҖӮиҝҷйҮҢд»…д»…жҳҜдёәдәҶжј”зӨәеҰӮдҪ•дҪҝз”ЁSPIFFS.format();гҖӮ

зЁӢеәҸз¬¬30иЎҢпјҢSPIFFS.begin()з”ЁдәҺеҗҜеҠЁй—Әеӯҳж–Үд»¶зі»з»ҹгҖӮеңЁжҜҸж¬ЎдҪҝз”Ёй—Әеӯҳж–Үд»¶зі»з»ҹд»ҘеүҚйғҪйңҖиҰҒжү§иЎҢиҝҷдёҖж“ҚдҪңгҖӮеҰӮжһңй—Әеӯҳж–Үд»¶зі»з»ҹжҲҗеҠҹеҗҜеҠЁпјҢиҜҘеҮҪж•°зҡ„иҝ”еӣһеҖјдёәеёғе°”еһӢпјҢеҰӮжһңжҲҗеҠҹеҗҜеҠЁй—Әеӯҳж–Үд»¶еҪўеҗҢпјҢеҲҷиҝ”еӣһзңҹгҖӮеҗҰеҲҷе°Ҷиҝ”еӣһеҒҮгҖӮ

зЁӢеәҸз¬¬36иЎҢпјҢFile dataFile = SPIFFS.open(file_name, "w");иҝҷжқЎиҜӯеҸҘдёӯпјҢopenеҮҪж•°еҸҜз”ЁдәҺеҜ№SPIFFSиҝӣиЎҢж“ҚдҪңгҖӮиҜҘеҮҪж•°е…ұжңүдёӨдёӘеҸӮж•°гҖӮз¬¬дёҖдёӘеҸӮж•°file_nameжҳҜиў«ж“ҚдҪңзҡ„ж–Үд»¶еҗҚз§°пјҢжң¬зӨәдҫӢдёӯиҜҘж–Үд»¶дёә/taichi-maker/notes.txt
з¬¬дәҢдёӘеҸӮж•°"w"д»ЈиЎЁжӯӨж“ҚдҪңдёәеҗ‘SPIFFSеҶҷе…Ҙж–Үд»¶дҝЎжҒҜгҖӮиҜ·жіЁж„ҸпјҡеҰӮжһңж–Үд»¶зі»з»ҹжІЎжңү/taichi-maker/notes.txtж–Үд»¶пјҢжӯӨж“ҚдҪңе°ҶдјҡеңЁж–Үд»¶зі»з»ҹдёӯе»әз«ӢиҜҘж–Үд»¶гҖӮеҰӮжһңж–Үд»¶зі»з»ҹжңүиҜҘж–Үд»¶пјҢеҲҷзЁӢеәҸе°ҶдјҡйҮҚж–°е»әз«ӢиҜҘж–Үд»¶пјҢеҚіеҺҹжңүж–Үд»¶дҝЎжҒҜе°Ҷдјҡиў«иҰҶзӣ–гҖӮ

зЁӢеәҸз¬¬37иЎҢdataFile.println("Hello IOT World.");з”ЁдәҺеҗ‘dataFileж–Үд»¶еҶҷе…ҘдҝЎжҒҜгҖӮдҝЎжҒҜеҶ…е®№дёәвҖңHello IOT World.вҖқгҖӮ

зЁӢеәҸз¬¬38иЎҢdataFile.close();з”ЁдәҺе…ій—ӯdataFileж–Үд»¶гҖӮз»“жқҹж–Үд»¶ж“ҚдҪңеҗҺпјҢеә”жү§иЎҢжӯӨж“ҚдҪңгҖӮ

2гҖҒйҖҡиҝҮзЁӢеәҸд»Һй—Әеӯҳж–Үд»¶зі»з»ҹиҜ»еҸ–дҝЎжҒҜ

д»ҘдёҠзЁӢеәҸз¬¬36иЎҢпјҢSPIFFS.exists(file_name)з”ЁдәҺжЈҖжҹҘй—Әеӯҳж–Үд»¶зі»з»ҹдёӯжңүfile_nameж–Үд»¶пјҲжіЁпјҡfile_nameеҸҳйҮҸе…·дҪ“дҝЎжҒҜеңЁжң¬зЁӢеәҸз¬¬23иЎҢпјүгҖӮиҜҘеҮҪж•°иҝ”еӣһеҖјдёәеёғе°”еһӢгҖӮеҰӮжһңж–Үд»¶еӯҳеңЁеҲҷиҝ”еӣһзңҹпјҢеҗҰеҲҷе°Ҷиҝ”еӣһеҒҮгҖӮ

д»ҘдёҠзЁӢеәҸз¬¬45иЎҢпјҢFile dataFile = SPIFFS.open(file_name, "r");иҝҷжқЎиҜӯеҸҘдёӯпјҢopenеҮҪж•°еҸҜз”ЁдәҺеҜ№SPIFFSиҝӣиЎҢж“ҚдҪңгҖӮиҜҘеҮҪж•°е…ұжңүдёӨдёӘеҸӮж•°гҖӮз¬¬дёҖдёӘеҸӮж•°file_nameжҳҜиў«ж“ҚдҪңзҡ„ж–Үд»¶еҗҚз§°пјҢжң¬зӨәдҫӢдёӯиҜҘж–Үд»¶дёә/taichi-maker/notes.txt
з¬¬дәҢдёӘеҸӮж•°"r"д»ЈиЎЁжӯӨж“ҚдҪңдёәиҜ»еҸ–ж–Үд»¶дҝЎжҒҜгҖӮ

д»ҘдёҠзЁӢеәҸз¬¬48иЎҢзҡ„forеҫӘзҺҜиҜӯеҸҘдёӯпјҢеҫӘзҺҜжқЎд»¶дҪҝз”ЁдәҶеҮҪж•°dataFile.size()гҖӮиҜҘеҮҪж•°е°Ҷдјҡиҝ”еӣһdataFileзҡ„еӨ§е°ҸгҖӮеҫӘзҺҜиҜӯеҸҘдҪ“дёӯпјҢdataFile.read()е°ҶдјҡиҜ»еҸ–dataFileж–Үд»¶еҶ…е®№гҖӮжҜҸи°ғз”ЁдёҖж¬ЎиҜҘеҗ«зЁҺйғҪдјҡиҝ”еӣһdataFileж–Үд»¶дёӯдёҖдёӘеӯ—з¬ҰгҖӮеҶҚж¬Ўи°ғз”ЁпјҢе°Ҷдјҡиҝ”еӣһдёӢдёҖдёӘеӯ—з¬ҰгҖӮд»ҘжӯӨзұ»жҺЁпјҢзӣҙеҲ°dataFileз»“е°ҫгҖӮйҖҡиҝҮforеҫӘзҺҜиҜӯеҸҘпјҢзЁӢеәҸе°Ҷдјҡдҫқж¬ЎиҜ»еҸ–dataFileж–Үд»¶еҶ…е®№пјҢе№¶дё”е°Ҷж–Үд»¶еҶ…е®№йҖҗеӯ—з¬Ұиҫ“еҮәдәҺдёІеҸЈзӣ‘и§ҶеҷЁдёӯгҖӮ

3гҖҒйҖҡиҝҮзЁӢеәҸеҗ‘й—Әеӯҳж–Үд»¶зі»з»ҹж–Үд»¶ж·»еҠ дҝЎжҒҜ

еңЁд»ҘдёҠзЁӢеәҸзҡ„з¬¬40иЎҢйҮҢпјҢFile dataFile = SPIFFS.open(file_name, "a");иҝҷжқЎиҜӯеҸҘдёӯпјҢopenеҮҪж•°еҸҜз”ЁдәҺеҜ№SPIFFSиҝӣиЎҢж“ҚдҪңгҖӮиҜҘеҮҪж•°е…ұжңүдёӨдёӘеҸӮж•°гҖӮз¬¬дёҖдёӘеҸӮж•°file_nameжҳҜиў«ж“ҚдҪңзҡ„ж–Үд»¶еҗҚз§°,з¬¬дәҢдёӘеҸӮж•°"a"д»ЈиЎЁеҗ‘иҜҘж–Үд»¶ж·»еҠ дҝЎжҒҜгҖӮиҜ·з•ҷж„ҸпјҢжӯӨеӨ„зҡ„ж·»еҠ дҝЎжҒҜжҳҜдёҚдјҡеҲ йҷӨж–Үд»¶еҶ…еҺҹжңүдҝЎжҒҜпјҢиҖҢжҳҜеңЁеҺҹжңүдҝЎжҒҜеҗҺйқўж·»еҠ ж–°зҡ„дҝЎжҒҜгҖӮиҝҷдёҺдҪҶеҶҷе…Ҙж“ҚдҪңжҳҜжңүжүҖеҢәеҲ«зҡ„гҖӮеҶҷе…Ҙж“ҚдҪңжҳҜе°Ҷж–Үд»¶еҶ…е®№е®Ңе…Ёжё…йҷӨпјҢйҮҚж–°еҶҷе…Ҙж–°дҝЎжҒҜгҖӮ
,
зЁӢеәҸзҡ„з¬¬41иЎҢпјҢdataFile.println("This is Appended Info.")пјҢжӯӨиҜӯеҸҘдҪңз”Ёе°Ҷдјҡеҗ‘dataFileж–Үд»¶е°ҫйғЁж·»еҠ еҸҢеј•еҸ·дёӯзҡ„дҝЎжҒҜеҶ…е®№пјҢд№ҹе°ұжҳҜеңЁж–Үд»¶е°ҫйғЁж·»еҠ вҖңThis is Appended Info.вҖқгҖӮ

4гҖҒйҖҡиҝҮзЁӢеәҸиҜ»еҸ–зӣ®еҪ•еҶ…е®№

жң¬зЁӢеәҸз¬¬23иЎҢе»әз«ӢдәҶдёҖдёӘеӯ—з¬ҰдёІеҸҳйҮҸгҖӮиҜҘеҸҳйҮҸз”ЁдәҺеӯҳеӮЁж–Үд»¶еӨ№еҗҚгҖӮе…¶дёӯ/д»ЈиЎЁж №зӣ®еҪ•гҖӮ/taichi-maker/д»ЈиЎЁж №зӣ®еҪ•дёӢзҡ„taichi-makerзӣ®еҪ•гҖӮ

жң¬зЁӢеәҸз¬¬41иЎҢSPIFFS.openDir(folder_name)дёӯзҡ„openDirеҮҪж•°еҮҪе°Ҷиҝ”еӣһдёҖдёӘвҖңзӣ®еҪ•вҖқеҜ№иұЎе№¶дё”иөӢеҖјз»ҷdirгҖӮжӯӨвҖқзӣ®еҪ•вҖқеҜ№иұЎжӯЈжҳҜfolder_nameжүҖеӯҳеӮЁзҡ„/taichi-maker/зӣ®еҪ•гҖӮеҗҺз»ӯзЁӢеәҸеҜ№dirзҡ„жүҖжңүж“ҚдҪңйғҪжҳҜй’ҲеҜ№/taichi-maker/жүҖжү§иЎҢзҡ„гҖӮ

жң¬зЁӢеәҸз¬¬43иЎҢwhileеҫӘзҺҜиҜӯеҸҘзҡ„еҫӘзҺҜжқЎд»¶жҳҜdir.next()зҡ„иҝ”еӣһеҖјгҖӮdir.next()еҮҪж•°з”ЁдәҺжЈҖжҹҘdirж–Үд»¶еӨ№еҶ…зҡ„ж–Үд»¶гҖӮжҲ‘д»¬еҸҜд»ҘжғіиұЎdirж–Үд»¶еӨ№йҮҢжңүдёҖдёӘжҢҮй’ҲпјҢжҜҸдёҖж¬Ўи°ғз”ЁnextеҮҪж•°йғҪдјҡи®©жҢҮй’Ҳеҗ‘дёӢжҢӘеҠЁдёҖж јгҖӮжҜҸдёҖж¬ЎжҢӘеҠЁдёҖж јпјҢеҰӮжһңдёӢдёҖдёӘдҪҚзҪ®жңүж–Үд»¶пјҢеҲҷиҝ”еӣһзңҹгҖӮеҗҰеҲҷе°Ҷдјҡиҝ”еӣһеҒҮгҖӮеӣ жӯӨпјҢwhile (dir.next())еҫӘзҺҜиҜӯеҸҘдёӯзҡ„еҶ…е®№дјҡдҫқж¬ЎжҳҫзӨәdirж–Үд»¶еӨ№дёӯзҡ„жҜҸдёҖдёӘж–Үд»¶зҡ„ж–Үд»¶еҗҚгҖӮ

5гҖҒд»Һй—Әеӯҳж–Үд»¶зі»з»ҹдёӯеҲ йҷӨж–Үд»¶

д»ҘдёҠзЁӢеәҸдёӯз¬¬25иЎҢпјҢйҖҡиҝҮдҪҝз”ЁSPIFFS.remove(file_name)е°Ҷfile_nameжүҖжҢҮд»Јзҡ„ж–Үд»¶иҝӣиЎҢдәҶеҲ йҷӨж“ҚдҪңгҖӮеҸҰеӨ–пјҢSPIFFS.remove(file_name)зҡ„иҝ”еӣһеҖјдёәеёғе°”еһӢгҖӮеҰӮжһңж–Үд»¶еҲ йҷӨжү§иЎҢжҲҗеҠҹеҲҷиҝ”еӣһзңҹпјҢеҗҰеҲҷиҝ”еӣһеҒҮгҖӮ

6гҖҒжҳҫзӨәй—Әеӯҳж–Үд»¶зі»з»ҹдҝЎжҒҜ

д»ҘдёҠзЁӢеәҸз¬¬16иЎҢиҜӯеҸҘFSInfo fs_info;е»әз«ӢдәҶFSInfo еҜ№иұЎпјҢз”ЁдәҺеӯҳеӮЁй—ӘеӯҳзҠ¶жҖҒдҝЎжҒҜгҖӮ

д»ҘдёҠзЁӢеәҸз¬¬26иЎҢиҜӯеҸҘSPIFFS.info(fs_info);гҖӮйҖҡиҝҮinfoеҮҪж•°е°Ҷй—ӘеӯҳзҠ¶жҖҒдҝЎжҒҜиөӢз»ҷfs_infoгҖӮеҗҺз»ӯзҡ„зЁӢеәҸдёӯпјҢйҖҡиҝҮдёҖзі»еҲ—иҜӯеҸҘе°Ҷй—ӘеӯҳзҠ¶жҖҒдҝЎжҒҜйҖҡиҝҮдёІеҸЈзӣ‘и§ҶеҷЁиҫ“еҮәгҖӮе…·дҪ“дҝЎжҒҜеҶ…е®№еҸҜеҸӮиҖғзЁӢеәҸжіЁйҮҠйғЁеҲҶгҖӮ

д»ҘдёҠжҳҜе…ідәҺESP8266й—Әеӯҳж–Үд»¶зі»з»ҹзҡ„еёёз”ЁеҠҹиғҪд»Ӣз»ҚгҖӮе…ідәҺESP8266й—Әеӯҳж–Үд»¶зі»з»ҹзҡ„жӣҙеӨҡж“ҚдҪңд»Ӣз»ҚпјҢиҜ·еҸӮиҖғArduino ESP8266е®ҳж–№йЎөйқўдёӯзҡ„д»Ӣз»ҚйғЁеҲҶгҖӮиҜҘйЎөйқўеҸҜзӮ№еҮ»д»ҘдёӢй“ҫжҺҘеүҚеҫҖгҖӮ

https://arduino-esp8266.readthedocs.io/en/latest/filesystem.html

дәҢгҖҒйҖҡиҝҮArduino IDEеҗ‘й—Әеӯҳж–Үд»¶зі»з»ҹдёҠдј ж–Үд»¶

1гҖҒдёӢиҪҪ Arduino-ESP8266й—Әеӯҳж–Үд»¶жҸ’д»¶зЁӢеәҸ

жӮЁжңүдёӨз§Қж–№жі•еҸҜд»ҘдёӢиҪҪиҜҘжҸ’д»¶зЁӢеәҸгҖӮ

з¬¬дёҖз§Қж–№жі•жҳҜйҖҡиҝҮзӮ№еҮ»жӯӨй“ҫжҺҘиҝӣе…Ҙ Arduino-ESP8266е®ҳж–№GitHubйЎөйқўдёӢиҪҪгҖӮиҜҰжғ…иҜ·и§Ғд»ҘдёӢжҲӘеӣҫпјҡ

з”ұдәҺGitHubзҪ‘з«ҷжңҚеҠЎеҷЁеңЁеӣҪеӨ–пјҢжңүдәӣжңӢеҸӢзҷ»еҪ•иҜҘзҪ‘з«ҷдјҡжңүдәӣеӣ°йҡҫгҖӮеҰӮжһңжҳҜиҝҷж ·зҡ„иҜқпјҢйӮЈд№ҲжӮЁд№ҹеҸҜд»ҘзӮ№еҮ»жӯӨй“ҫжҺҘиҝӣе…ҘеӨӘжһҒеҲӣе®ўзҪ‘з«ҷдёӢиҪҪйЎөйқўеҺ»дёӢиҪҪиҜҘжҸ’д»¶гҖӮ

2гҖҒзЎ®е®ҡArduino IDEйЎ№зӣ®ж–Үд»¶еӨ№дҪҚзҪ®

3гҖҒйҖҡиҝҮиө„жәҗз®ЎзҗҶеҷЁжү“ејҖArduino IDEйЎ№зӣ®ж–Үд»¶еӨ№е№¶е»әз«Ӣtoolsж–Үд»¶еӨ№

4гҖҒи§ЈеҺӢзј©esp8266fsжҸ’д»¶еҺӢзј©еҢ…пјҢе№¶е°Ҷи§ЈеҺӢзј©еҗҺзҡ„ж–Үд»¶еҶ…е®№зІҳиҙҙеҲ°йЎ№зӣ®ж–Үд»¶еӨ№дёӯ

5гҖҒйҮҚж–°еҗҜеҠЁArduino IDE

6гҖҒж №жҚ®дёҠдј зҡ„ж–Үд»¶жҖ»еӨ§е°ҸжқҘи®ҫзҪ®й—ӘеӯҳеӨ§е°Ҹ

8гҖҒе°ҶйңҖиҰҒдёҠдј зҡ„ж–Үд»¶дҝқеӯҳеңЁзЁӢеәҸи·Ҝеҫ„дёӢзҡ„dataж–Үд»¶еӨ№дёӯ
